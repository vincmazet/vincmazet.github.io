Code de Huffman

Le code de Huffman (1952) est un type de codage source obtenu à partir des probabilités d'occurrence des symboles. Ce code est de longueur variable et sa longueur moyenne est la plus petite parmi tous les codes à décodage unique et instantané.

On dispose de \(N\) symboles dénomés \(S_1\), ..., \(S_N\) ici (leur nom est inutile pour l'algorithme). Donnez les probabilités des symboles dans la zone de texte ci-dessous (en les séparant par une espace), puis cliquez sur « Calculer le code » ! Le code de Huffman s'affiche ensuite à gauche des symboles. Notez bien que le code de Huffman n'est pas unique.
